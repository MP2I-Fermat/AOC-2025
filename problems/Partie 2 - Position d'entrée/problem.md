Pour cette prochaine étape, votre programme devra lire l'entrée, puis afficher
**la position d'entrée du faisceau dans le DAOA**.

Par exemple, pour l'exemple précédent, l'entrée du DAOA est sur la **7e**
colonne (en comptant à partir de 0). Pour cette entrée, votre programme devra
alors afficher&nbsp;:
```
7
```

_Remarque : il n'est pas nécessaire pour cette partie de lire l'entrée en
entier. Seule la première ligne indique une position d'entrée._
